# Burnout Tracker

A DBMS-based web application that tracks student study patterns and detects burnout using backend logic.

## Features
- Add daily study logs
- Detect burnout based on sleep patterns
- REST API using Node.js
- MySQL database integration

## Tech Stack
- Node.js
- Express.js
- MySQL

## How it Works
The system analyzes the last 3 days of sleep data. If sleep is consistently below 6 hours, it flags burnout risk.# burnout-tracker
